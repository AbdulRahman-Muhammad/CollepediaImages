Search Tips & Tricks

Our search engine supports a powerful query syntax. Here are some examples to get you started.

Basic Search (AND logic)

Simply type words separated by spaces. This will find images that contain ALL of the specified tags.

blue car night

OR Logic

Use the `OR` operator (must be uppercase) to find images that match any of the terms.

cat OR dog sky (blue OR grey)

Excluding Tags

Use a hyphen (`-`) before a tag to exclude any images containing that tag.

animal -dog

Filtering by Owner

Use the `owner:` prefix to find images from a specific user.

owner:john car

Wildcard Search

Use an asterisk (`*`) as a wildcard to match parts of tags.

land*

Sorting Results

You can sort results by date using the `sort:` operator.

sort:date-desc cat sort:date-asc

Combining Operators

You can combine all operators to create highly specific queries.

owner:jane (cat OR dog) -black sort:date-desc
